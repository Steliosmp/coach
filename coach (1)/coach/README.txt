README –ΒΟΗΘΟΣ ΠΡΟΠΟΝΗΤΗ (Μέρος 1)

Η εφαρμογή υλοποιήθηκε σε Java και αφορά το Μέρος 1 της εργασίας.
Εκτελείται μέσω κονσόλας και αναλύει αρχεία αθλητικών δραστηριοτήτων τύπου .tcx.

Μεταγλώττιση:
mvn compile

Εκτέλεση:
java -cp target/classes org.example.App <αρχεία.tcx>

Παράδειγμα:
java -cp target/classes org.example.App run1.tcx

Η εφαρμογή διαβάζει ένα ή περισσότερα αρχεία .tcx και εμφανίζει βασικά
στατιστικά για κάθε δραστηριότητα, όπως συνολικό χρόνο, απόσταση,
μέσο ρυθμό ή ταχύτητα και μέσους καρδιακούς παλμούς (αν υπάρχουν).

Προαιρετικά μπορεί να δοθεί βάρος με το flag -w, οπότε εμφανίζονται
και οι θερμίδες.

Παράδειγμα:
java -cp target/classes org.example.App -w 65.9 run1.tcx
